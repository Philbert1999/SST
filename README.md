# SST - A 股短线候选池筛选器

SST 是一个用 Python + Streamlit 编写的 A 股短线候选池筛选工具。

它的目标不是自动交易，而是帮助你：

- 每天批量筛选短线候选股
- 查看候选股触发了哪些策略
- 记录每天筛选出来的股票池
- 过几天后复盘这些股票未来 5 个交易日的收益表现
- 直观看到筛选策略的质量

本项目仅用于学习、研究和辅助观察，不提供任何自动下单功能。

## 主要功能

### 候选池筛选

- 使用 AKShare 获取 A 股股票列表和日线数据
- 支持多个短线策略同时运行
- 同一只股票触发多个策略时自动合并得分
- 按 `total_score` 从高到低排序
- 显示触发策略、触发原因、风险提示
- 支持导出 CSV

### 基础过滤

当前支持的过滤条件包括：

- 排除 ST / *ST
- 排除退市风险股票
- 排除上市不足 60 天的新股
- 排除科创板股票
- 最低股价
- 最高股价
- 最低成交额
- 排除最近 20 日涨幅过高的过热股票

### 策略

当前主要策略包括：

- 放量突破
- 强势回踩
- 强度打分

策略只输出候选信号和评分，不给出“必买”结论。

### K 线图

页面中可以查看候选股 K 线图，包括：

- K 线
- MA5
- MA10
- MA20
- 成交量

### 筛选质量复盘

这是项目新增的重点功能。

每次运行筛选后，如果有候选结果，系统会自动保存一份筛选快照到：

```text
snapshots/
```

之后可以在页面的 **效果复盘** 页签中选择某次历史筛选结果，并计算这些股票在未来 5 个交易日的表现：

- D1 收益率
- D2 收益率
- D3 收益率
- D4 收益率
- D5 收益率
- 5 日内最高收益
- 5 日内最低收益
- D1-D5 平均收益
- D1-D5 上涨占比

收益计算方式：

```text
筛选日收盘价 -> 未来第 N 个交易日收盘价
```

这个功能可以帮助你判断：每天筛出来的股票，短期内到底有没有上涨，平均涨了多少，胜率怎么样。

## 安装

建议使用 Python 3.10 或更高版本。

```bash
git clone https://github.com/Philbert1999/SST.git
cd SST
pip install -r requirements.txt
```

如果你是直接在本地项目目录中运行：

```bash
cd D:\Philbert\Documents\stock_short_term
pip install -r requirements.txt
```

## 运行

```bash
streamlit run app.py
```

启动后打开：

```text
http://localhost:8501
```

页面中有两个主要页签：

- 候选筛选：运行策略筛选股票
- 效果复盘：查看历史筛选结果未来 5 日表现

## 使用流程

1. 打开 Streamlit 页面。
2. 在侧边栏设置日期范围、策略、最低成交额、最低股价、最高股价等条件。
3. 点击 **运行筛选**。
4. 查看候选池结果和 K 线图。
5. 系统会自动保存本次候选池快照。
6. 过几天后进入 **效果复盘** 页签。
7. 选择历史快照，点击 **计算未来 5 个交易日表现**。
8. 查看 D1-D5 收益、上涨占比和明细表。

## 数据源说明

项目使用 AKShare 获取数据。

当前代码中使用了多个接口，并带有备用逻辑：

- 股票列表：优先实时行情接口，失败后使用股票代码列表接口
- 日线数据：优先东方财富日线接口，失败后使用新浪日线接口
- 板块数据：使用东方财富板块接口

如果你遇到股票列表为空、日线数据为空、运行很慢等问题，通常和网络、代理、防火墙或 AKShare 底层接口有关。

## 配置

主要参数集中在 `config.py` 中。

常用配置包括：

- `min_price`：最低股价
- `max_price`：最高股价
- `min_amount`：最低成交额
- `exclude_st`：是否排除 ST
- `exclude_new_stock_days`：新股过滤天数
- `exclude_star_market`：是否排除科创板
- `max_20d_gain`：最近 20 日最大涨幅过滤

页面侧边栏中的默认值也来自 `config.py`。

## 注意事项

- 第一次运行会比较慢，因为需要请求股票列表和日线数据
- 项目已经加入文件缓存，第二次运行通常会快一些
- 免费数据源可能不稳定，接口可能延迟、失败或字段变化
- 建议先缩小筛选范围，比如提高最低成交额、限制最高股价
- 复盘结果依赖后续日线数据，如果未来交易日还没发生，对应收益会为空
